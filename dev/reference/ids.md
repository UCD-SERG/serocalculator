# Get person IDs

Get person IDs

## Usage

``` r
ids(object, ...)
```

## Arguments

- object:

  a [`data.frame()`](https://rdrr.io/r/base/data.frame.html)

- ...:

  unused

## Value

a [character](https://rdrr.io/r/base/character.html)
[vector](https://rdrr.io/r/base/vector.html)
