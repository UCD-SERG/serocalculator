# Extract `Strata` metadata from an object

Generic method for extracting strata metadata from objects. See
[`strata.default()`](https:/ucd-serg.github.io/serocalculator/preview/pr462/reference/strata.default.md)

## Usage

``` r
strata(x)
```

## Arguments

- x:

  an object

## Value

the strata metadata of `x`
