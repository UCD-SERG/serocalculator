# generate random sample from baseline distribution

generate random sample from baseline distribution

## Usage

``` r
mk_baseline(kab, n = 1, blims, ...)
```

## Arguments

- kab:

  [integer](https://rdrr.io/r/base/integer.html) indicating which row to
  read from `blims`

- n:

  number of observations

- blims:

  range of possible baseline antibody levels

- ...:

  not currently used

## Value

a [`numeric()`](https://rdrr.io/r/base/numeric.html) vector
