# Plot download data with adaptive faceting

Plot download data with adaptive faceting

## Usage

``` r
# S3 method for class 'download_data'
autoplot(object, ...)
```

## Arguments

- object:

  A `download_data` object from
  [`.get_download_data()`](https://ucd-serg.github.io/serocalculator/reference/dot-get_download_data.md).

- ...:

  Not currently used.

## Value

A
[`ggplot2::ggplot()`](https://ggplot2.tidyverse.org/reference/ggplot.html)
object.
